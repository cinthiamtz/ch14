6. Elabora un algoritmo para leer un número y determinar si es par o impar.

  • Declare una función llamada isPairNumber con un parámetro llamado A.
  • Coloque un if con un condicional que imprime si el número ingresado es par.
  • Si el número es par imprime "El número ingresado es par”.
  • De lo contrario si el número es impar imprime "El número ingresado no es par".
