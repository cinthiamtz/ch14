5. Elaborar un algoritmo para leer 3 números y determinar sí uno es la suma de los otros dos

  • Declare una función llamada addition con un parámetro (A, B, C ).
  • Coloque condicionales if y else if con comparaciones.
  • En la primera condición if comparo si A es igual a B + C imprime "La suma de B y C da el mismo resultado de A".
  • En la segunda condición else if comparó si B es igual a A + C imprime "La suma de A y C da el mismo resultado de B".
  • En la tercera condición else if comparó si C es igual a A + B imprime "La suma de A y B da el mismo resultado de C".
  • De lo contrario si no es igual a ninguna de las opciones imprime ("No se encontró ninguna coincidencia en las sumas".
